# Spendify 💸

A simple and clean budget tracking web app built with HTML, CSS, and JavaScript.

## 🚀 Features
- Set monthly budget (KES)
- Add and delete expenses
- Track total spent and remaining balance
- Category-based expense tracking
- Visual spending chart (Chart.js)

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js

## 📦 How to Use
1. Enter your monthly budget
2. Add your expenses
3. View your spending summary and chart
